# ec2_start-stop

Start and stop ec2 instances with slack

## Environment

- Slack
- EC2
- Api Gateway
- Lambda (node8.10)
- Slash command

## use

<img width="775" alt="スクリーンショット_2019-05-20_9_30_34" src="https://user-images.githubusercontent.com/5633085/57990540-812e3f00-7ae2-11e9-97ea-57882e4ab007.png">

- status

```
/ec2 dev1 status
```
- start

```
/ec2 dev1 start
```
- stop

```
/ec2 dev1 stop
```
